# ESP32-C6: Sincronização de Tempo via Wi-Fi e SNTP

Este projeto implementa um firmware para o microcontrolador ESP32-C6 utilizando o framework ESP-IDF. O sistema conecta-se a uma rede Wi-Fi (modo Station) e sincroniza a data e hora atual utilizando os servidores NTP brasileiros (`ntp.br`). O fuso horário é configurado de forma fixa para o Horário de Brasília (UTC-3).
